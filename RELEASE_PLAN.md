# Release Plan

**v0.1-interim (this release).** Completed pre-registered arms + harness + results, with the
open boundary explicit (STATUS.md). Tagged as a GitHub prerelease. No claim is upgraded past
its evidence; nulls are shown.

**Two publication moments (deliberate):**
1. *Now* — interim empirical result + the context-governance thesis. Repo is audit-ready:
   raw rows, harness, specs, and rerun commands present so wave-2 audiences can be answered,
   not merely promised.
2. *Later* — the complete battery (Arm 3 Stage 2, authority redesign, broader provider
   coverage), full raw evidence package, and a DOI release (Zenodo on the final GitHub tag).

**What will NOT happen:** retroactively labeling completed arms as OSF-preregistered. Freeze
was enforced by in-file timestamps + content-hash registries; that is what we claim. Stage 2's
protocol may be publicly registered *before* its data is analyzed (timestamped, read-only).

**Distribution waves.** Wave 1 (own site canonical + this repo + LinkedIn). Wave 2, once the
repo has more raw analysis material: LessWrong / Alignment Forum / HN / r/ML — audiences that
will ask for prompts, exclusions, and rows; the repo must answer.
